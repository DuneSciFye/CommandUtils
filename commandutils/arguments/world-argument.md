# World Argument

The World Argument requires the name of a Bukkit world, such as `world`. The world names are the same as the level folder names.

If you have a world manager plugin such as [Multiverse-Core](https://www.spigotmc.org/resources/multiverse-core.390/), you can use that to find all your world names.

{% hint style="info" %}
World names should be auto suggested for tab completion
{% endhint %}
