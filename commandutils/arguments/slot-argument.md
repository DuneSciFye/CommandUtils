# Slot Argument

Numerical numbers can be used, 0-40, and some keywords as well.

<figure><img src="../../.gitbook/assets/image (1).png" alt=""><figcaption><p>Numbered Inventory Slots from <a href="https://docs.ssomar.com/tools-for-all-plugins-score/general-questions-or-guides/utilities#slots">Ssomar Plugins</a></p></figcaption></figure>



| Keyword  | Slot                                     |
| -------- | ---------------------------------------- |
| main     | The item the player is currently holding |
| mainhand | The item the player is currently holding |
| off      | The item in the player's offhand         |
| offhand  | The item in the player's offhand         |
| cursor   | The item in the player's cursor          |
| -1       | Mainhand slot, same as main              |
