# Place Block From Inv

Usage: /placeblockfrominv \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \<Material> \[\<Consume from Inventory>] \[\<Trigger Block Place Event>]

* World - The world the location is in
* Location - Coordinates of the block/location
* Material - The material to use
* Consume from Inventory _(optional)_ - Whether to consume a matching item from the inventory
* Trigger Block Place Event _(optional)_ - Whether to fire a BlockPlaceEvent
