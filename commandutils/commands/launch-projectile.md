# Launch Projectile

Usage: /launchprojectile \<Projectile> \[\<Max Alive Time>] \[\<Particle>] \[\<Period>]

Usage: /launchprojectile \<Projectile> \[\<Max Alive Time>] \[\<Initial Delay>] \[\<Period>] \[\<Command Separator>] \[\<Commands>]

Usage: /launchprojectile \<Projectile> \<Max Alive Time> \<Command Separator> \<Commands>

* Projectile - The projectile type to launch
* Max Alive Time _(optional)_ - Maximum time the projectile stays alive
* Particle _(optional)_ - The particle to display
* Period _(optional)_ - Ticks between each run
* Initial Delay _(optional)_ - Delay before the first run
* Command Separator _(optional)_ - The string used to separate individual commands
* Commands _(optional)_ - Commands to run, separated by the command separator
