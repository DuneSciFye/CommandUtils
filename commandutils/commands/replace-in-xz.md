# Replace In XZ

Usage: /replaceinxz \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \<Player> \<X> \<Z> \<Whitelisted Blocks> \<Material List> \[\<Apply Physics>] \[\<Duration>]

* World - The world the location is in
* Location - Coordinates of the block/location
* Player - The player to affect
* X - Size along the X axis
* Z - Size along the Z axis
* Whitelisted Blocks - Block predicate / whitelist of blocks that may be affected. Accepts a config-defined predicate name
* Material List - List of materials
* Apply Physics _(optional)_ - Whether to apply block physics updates
* Duration _(optional)_ - Duration, e.g. 5s, 1m, 100t
