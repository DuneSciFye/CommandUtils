# Waterlog

Usage: /waterlog \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \[\<Waterlogged State>] \[\<Radius>]

* World - The world the location is in
* Location - Coordinates of the block/location
* Waterlogged State _(optional)_ - Whether the block is waterlogged
* Radius _(optional)_ - Radius of the effect
