# Break And Replant

Usage: /breakandreplant \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \<Player> \<Radius> \<Original Block>

Usage: /breakandreplant \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \<Player> \<X> \<Y> \<Z> \<Original Block>

Usage: /breakandreplant \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \<Player> \<Original Block>

* World - The world the location is in
* Location - Coordinates of the block/location
* Player - The player to affect
* Radius - Radius of the effect
* Original Block - The block type that was originally broken
* X - Size along the X axis
* Y - Size along the Y axis
* Z - Size along the Z axis
