# Item Lore

Usage: /itemlore \<append | set | remove> \<Player> \<[slot](../arguments/slot-argument.md)> \<Line> \<Content>

* Player - The player to affect
* Slot - Slot of the item
* Line - Line number of the lore (starting at 1)
* Content - Value to set
