# Weighted Random

Usage: /weightedrandom \<cache> \<ID> \<Command Separator> \<Placeholder Surrounder> \<Arguments>

Usage: /weightedrandom \<run> \<Arguments>

Usage: /weightedrandom \<removecache> \<ID>

* ID - A unique identifier
* Command Separator - The string used to separate individual commands
* Placeholder Surrounder - The character used to surround placeholders (e.g. %)
* Arguments - The arguments / sub-commands to run (see examples)
