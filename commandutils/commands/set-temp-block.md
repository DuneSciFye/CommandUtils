# Set Temp Block

Usage: /settempblock \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \<Block State> \<Duration> \[\<Show Breaking>] \[\<Whitelisted Blocks>]

* World - The world the location is in
* Location - Coordinates of the block/location
* Block State - The block (and its block data) to use
* Duration - Duration, e.g. 5s, 1m, 100t
* Show Breaking _(optional)_ - Whether to show block-breaking animation
* Whitelisted Blocks _(optional)_ - Block predicate / whitelist of blocks that may be affected. Accepts a config-defined predicate name
