# Block Gravity

Usage: /blockgravity \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \[\<Gravity Enabled>] \[\<Radius>]

* World - The world the location is in
* Location - Coordinates of the block/location
* Gravity Enabled _(optional)_ - Whether the block should be affected by gravity
* Radius _(optional)_ - Radius of the effect
