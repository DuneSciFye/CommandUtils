# Add Item NBT

Usage: /additemnbt \<player> <[slot](../arguments/slot-argument.md)> \<namespace> \<key> \<amount>

* Player - Player whose inventory is used
* [Slot](../arguments/slot-argument.md) - Slot of Item
* Namespace - Namespace of NBT
* Key - Key of NBT
* Amount - Numerical value to add to NBT

Uses [NameSpacedKey](../arguments/namespacedkeys.md)
