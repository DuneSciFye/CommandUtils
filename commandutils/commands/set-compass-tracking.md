# Set Compass Tracking

Usage: /setcompasstracking \<Player> \<[slot](../arguments/slot-argument.md)> \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)>

Usage: /setcompasstracking \<Player> \<[slot](../arguments/slot-argument.md)> \<Target>

* Player - The player to affect
* Slot - Slot of the item
* World - The world the location is in
* Location - Coordinates of the block/location
* Target - The target entity/player
