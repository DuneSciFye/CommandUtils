# Replace In Radius If Block Relative

Usage: /replaceinradiusifblockrelative \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \<Player> \<Radius> \<Blocks To Replace From> \<Blocks To Replace To> \<Block Faces> \<Blocks Relative> \[\<Remove Item>]

Usage: /replaceinradiusifblockrelative \<[world](../arguments/world-argument.md)> \<[location](../arguments/block-location-argument.md)> \<Player> \<Radius> \<Blocks To Replace From> \<Block To Replace To> \<Block Faces> \<Blocks Relative> \[\<Remove Item>]

* World - The world the location is in
* Location - Coordinates of the block/location
* Player - The player to affect
* Radius - Radius of the effect
* Blocks To Replace From - Blocks that may be replaced
* Blocks To Replace To - Blocks to replace them with
* Block Faces - Which block faces to check for the relative block
* Blocks Relative - Blocks that must be present on the given faces
* Remove Item _(optional)_ - Item to remove from the player when a block is replaced
* Block To Replace To - The block (with block data) to replace matches with
