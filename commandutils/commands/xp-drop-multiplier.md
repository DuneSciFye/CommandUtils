# Xp Drop Multiplier

Usage: /xpdropmultiplier \<set> \<Players> \<Multiplier> \[\<Blacklist>]

Usage: /xpdropmultiplier \<clear> \<Players>

* Players - The players to affect (supports selectors)
* Multiplier - The multiplier to apply
* Blacklist _(optional)_ - Entity types to exclude
