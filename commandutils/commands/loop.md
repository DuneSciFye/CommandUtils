# Loop

Usage: /loop \<Function> \<Command ID> \<Loop Amount> \<Initial Delay> \<Period> \<Commands> \[\<End Commands>] \[\<Player>]

Usage: /loop \<run> \<Loop Amount> \<Initial Delay> \<Period> \<Commands> \[\<End Commands>] \[\<Player>]

* Function - Which sub-function to run
* Command ID - A unique identifier for the scheduled/looping task
* Loop Amount - Number of times to loop
* Initial Delay - Delay before the first run
* Period - Ticks between each run
* Commands - Commands to run, separated by the command separator
* End Commands _(optional)_ - Commands to run when the loop finishes
* Player _(optional)_ - The player to affect
