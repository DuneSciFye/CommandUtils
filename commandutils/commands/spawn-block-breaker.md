# Spawn Block Breaker

Usage: /spawnblockbreaker \<[location](../arguments/block-location-argument.md)> \[\<Yaw>] \[\<Pitch>] \[\<Item>] \[\<Vector Multiplier>] \[\<Radius>] \[\<Period>] \[\<Max Time>]

* Location - Coordinates of the block/location
* Yaw _(optional)_ - Yaw rotation
* Pitch _(optional)_ - Pitch rotation
* Item _(optional)_ - The item to use
* Vector Multiplier _(optional)_ - Multiplier applied to the launch vector
* Radius _(optional)_ - Radius of the effect
* Period _(optional)_ - Ticks between each run
* Max Time _(optional)_ - Maximum time the entity lives before despawning
