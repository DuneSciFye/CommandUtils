# Ray Trace Particle

Usage: /raytraceparticle \<Particle> \<Length> \<Spacing> \<Period> \<Entity> \[\<Commands>] \[\<Command Separator>] \[\<X Placeholder>] \[\<Y Placeholder>] \[\<Z Placeholder>]

* Particle - The particle to display
* Length - Length of the ray in blocks
* Spacing - Spacing between each particle
* Period - Ticks between each run
* Entity - The entity to affect
* Commands _(optional)_ - Commands to run, separated by the command separator
* Command Separator _(optional)_ - The string used to separate individual commands
* X Placeholder _(optional)_ - Placeholder substituted with the X coordinate
* Y Placeholder _(optional)_ - Placeholder substituted with the Y coordinate
* Z Placeholder _(optional)_ - Placeholder substituted with the Z coordinate
