# Set Item NBT

Usage: /setitemnbt \<Player> \<[slot](../arguments/slot-argument.md)> \<Namespace> \<Key> \[\<Content>]

* Player - The player to affect
* Slot - Slot of the item
* Namespace - Namespace of the NBT key
* Key - Key of the NBT
* Content _(optional)_ - Value to set

Uses [NameSpacedKey](../arguments/namespacedkeys.md)
