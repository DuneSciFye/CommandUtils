# Cooldown Command

Usage: /cooldowncommand \<Player> \<ID> \<Time> \<Commands> \[\<Command Separator>]

Usage: /cooldowncommand \<run2 | silent2> \<Player> \<ID> \<Time> \<Commands>

Usage: /cooldowncommand \<run | silent | global> \<Player> \<ID> \<Time> \<Commands> \[\<Command Separator>]

* Player - The player to affect
* ID - A unique identifier
* Time - Duration, e.g. 5s, 1m, 100t
* Commands - Commands to run, separated by the command separator
* Command Separator _(optional)_ - The string used to separate individual commands
